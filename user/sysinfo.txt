**15.SYSINFO**

Purpose:Command prints the system information

Status:Implemented

Syntax:

     sysinfo

Comments:

Command prints the useful system information which users may use to send bug reports to the gnucap developers community. This command doesn't works on Windows.

Example:

      gnucap> sysinfo
      Model name	      : Intel(R) Core(TM) i5-2450M CPU @ 2.50GHz
      OS Description    :"Ubuntu 12.04 LTS"
      CPU MHz	      : 2494.420
      RAM Available     : 1012348kB
      RAM Free          : 145640kB
      gnucap>
